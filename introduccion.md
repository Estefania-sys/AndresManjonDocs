# Bienvenido al Sistema de Biblioteca

Este proyecto utiliza **PHP 8** y **SQLite** para gestionar el catálogo de libros y préstamos escolares.

## Características principales
* Doble interfaz para alumnos (Infantil y Primaria).
* Gestión de usuarios por roles (Admin, Profe, Alumno).
* Base de datos portátil en un solo archivo `.db`.
